Merxcore

Merxcore develops software solutions and provides contracting services for development of websites, web platforms, and mobile applications.




Contracting

Merxcore has worked with clients to build simple websites, complex web services, and cross platform mobile apps. We offer private contracting to businesses or individuals in need of software solutions. Please email us to discuss any ideas or to receive a quote.



Past

Merxcore was founded in 2011 around the development of PHP software solutions. The first major project was Merxcore CRM, a Contact Relationship Management solution for small businesses.

Since 2011, we have been exploring best methods for developing cross-platform applications and have been building websites and web services for clients.

Present

Today Merxcore is focused on developing cutting edge software platforms and applications. We specialize in the Ruby on Rails framework for our backend solutions and Javascript for our frontend (Specifically Ember.js). With these languages we are able to build cross-platform, scalable solutions ranging from basic websites to complex platforms.





Specilizations

We specialize in building scalable, cross-platform software solutions with Javascript and Ruby. Other languages we specialize in include PHP, Python, C, Objective-C, C++, Java, and others. Our primary focus is in web and mobile solutions.
